      Over de app

      Deze app kan ik gebruiken voor mijn hobby's.

      Als ik ga fitnessen kan ik makkelijk zien welke oefeningen ik doe per spiergroep, en hoeveel gewicht ik vorige keer had. Als ik zwaarder kan gaan dan kan ik dit aanpassen. De nieuwe gegevens zullen opslaan bij het verlaten van de app.

      Als ik thuis met mijn instrumenten bezig ben dan kan ik door deze app terug wat inspiratie opdoen. Ik kan per instrument kijken welke liedjes ik kan spelen. Hierbij krijg ik soms ook een link naar een YouTube filmpje zodat ik mee kan spelen met het liedje. Ook kan ik zien welke liedjes ik nog wil leren spelen.
